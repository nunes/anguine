Anguine is template application. It is meant for building applications on a certain way. It is not a framework.

Based on a mixed architecture: both server side and client side MVC. Client side is intended to build a complete web application and has access to data through REST. Server side templating is intented for publishing meaningful urls and search engine indexing.
